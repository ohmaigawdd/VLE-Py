Introduction to Vapour Liquid Equilibrium (VLE):
Pure Component System

Learning Objectives

  • Visualize phase behaviour (VLE) of pure component system
  • Comprehend the changes in thermodynamic properties during phase transition
  • Recognize phase equilibrium criteria

Simulator Guidelines

  1. Choose system (water) and condition (T, P).
  2. Compress / expand the system isothermally (or) Heat / cool the system isobarically.
  3. As the system is subjected to the chosen isothermal or isobaric process, observe the molecular dynamics, changes in thermodynamic properties of the phases and the physical state of the system in the vapourization curve.

Self-Assessment

  • How would you conduct a bench scale investigation to obtain the vapourization curve?
  • Which thermodynamic property plays a vital role in defining the physical state of the system?
  • At a given system condition, how to identify the state of the pure species?
  • At a given system condition, how to infer the most stable phase?
  • How to comprehend whether the system is at phase equilibrium?
  • What happens when the system condition(s) are tweaked slightly at phase equilibrium?
